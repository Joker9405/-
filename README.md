# Can-Tong 粤语查询 · Cantonese Expression Helper

> 🗣️ 用普通话 / 英文 / 粤语输入，得到 **地道粤语写法 + 读音 + 例句解释**  
> 🌐 在线体验：**https://can-tong.vercel.app/**
> 🐦 X / Twitter：@oceanguest
我会在这里更新开发日志、词库进度、征集新词想法

Can-Tong 是一个专注「粤语本地用语」的小工具：  
你输入中文、英文，我们会尽量帮你匹配到正确的粤语写法、读音和简单解释，让你更了解地道粤语。

---

## 🚀 快速开始（给一般使用者）

👉 **最简单的使用方式：直接打开网站即可**

1. 打开：**https://can-tong.vercel.app/**
2. 在搜索框输入：
   - 普通话：`不好意思`、`失礼了`…
   - 英文：`so sorry`、`Excuse me`
3. 命中词条后，页面会展示：
   - ✅ **粤语写法（繁体）**
   - 🔊 **粤语读音**（点击播放）
   - 💬 **简单解释 + 用法**
   - 📌 相关例句（如果已有收录）
   - 粗口解锁用法

> 当前阶段是「MVP 公开测试」，词条数量还不多（正在慢慢长胖中 🍲）。  
> 如果没搜到你想要的词，欢迎帮我补充（见下文「如何贡献词条」）。

## ✨ 项目特点

- **三语视角**：围绕「普通话 / 粤语 / 英文理解」来解释一个表达，而不是只给你单一翻译。
- **真实语境**：词条尽量来自日常社交平台、影视对白、生活对话，而不是书面教科书。
- **无账号、无广告**：MVP 阶段完全免费使用，不需要登录，也没有广告打扰。
- **独立开发中**：由一人慢慢打磨，公开进展、接受建议，但词库质量严格把关。

---

## 🎯 适合谁？

- **学习粤语的人**：想学的不只是单词，而是「别人实际会怎么说」，「为什么这么说」。
- **在社交媒体发文的创作者**：想用粤语写 caption / 段子，又怕用错字/错语气。
- **海外华人 / 华侨**：想保持粤语、写得接近香港 / 粤语区的实际用法。
- **语言 / NLP 爱好者**：对粤语本地语料、表达演变感兴趣。

## 🙋‍♀️ 如何贡献词条？

目前众包阶段还比较原始，但已经可以简单参与：

### 方式一：通过公开表单 / 表格提交

> 📎 表单链接（示例）：  
> 👉 _[填入你的 Google Form / Sheet 链接](https://docs.google.com/spreadsheets/d/15kxXv_5WKHRtN8VJh86zGlMDrSntsJXXs_lFhd9wrpo/edit?usp=sharing)_

- 粤语本地说法（繁体）
- 对应英文理解 / 解释
- 例句 & 用法说明
- 如果是粗口，用勾选标记（方便我们以后加开关）

### 方式二：在 GitHub 开 issue

1. 打开本仓库的 Issues
2. 选择「新词条 / 例句建议」模板（如果有）
3. 填写你想补充的内容

> 所有贡献我都会认真看，每次小版本更新也会在 README / X 上感谢贡献者（可以匿名）。
